**flameshot虽然可以录屏，但是格式局限于webm，不利于在聊天工具上实时播放（比如飞书）。** flameshot在截图方面非常优秀，在ubuntu系统，依然还是强大且常用的工具。

安装：
``` bash
sudo add-apt-repository ppa:peek-developers/stable
sudo apt-get update
sudo apt install peek
```
